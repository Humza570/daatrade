# DaaTrade — Digital Trading Marketplace

A full-featured **B2B/B2C trading marketplace** built with Laravel and 
Livewire — connecting vendors and buyers with category-based listings, 
WhatsApp inquiry system, social authentication, and complete admin 
management panel.

## 🌐 Live

[daatrade.com](https://daatrade.com/)

## 🎯 Key Highlights

- 💬 **WhatsApp inquiry system** — buyer inquiries go directly to vendor 
  WhatsApp instantly
- 🔐 **Social login** — Google OAuth + Facebook Login
- 📝 **Livewire-powered blog** — admin manages blog posts with real-time 
  Livewire components
- 🏪 **Dual registration** — separate Vendor and Buyer registration flows
- 📧 **Subscriber management** — newsletter subscription system
- 🗂️ **Categories & sub-categories** — structured product browsing

## 👥 User Roles

| Role | Responsibility |
|---|---|
| **Admin** | Full platform control — categories, pages, blogs, users, subscribers |
| **Vendor** | Register, list products, manage inquiries |
| **Buyer** | Browse listings, send WhatsApp inquiries, register account |

## ✨ Features

### Marketplace
- Category and sub-category based product browsing
- Featured listings and search functionality
- Product detail pages with vendor information
- WhatsApp-based inquiry system — query goes directly to vendor WhatsApp

### Authentication
- Email + password registration and login
- Google OAuth integration
- Facebook Login integration
- Separate Vendor and Buyer registration flows

### Vendor System
- Vendor registration and profile management
- Product/listing management
- Inquiry tracking via WhatsApp
- Vendor dashboard

### Admin Panel
- Analytics dashboard
- Category and sub-category management
- Page management — dynamic pages via admin
- Blog management with Livewire real-time components
- Subscriber management — newsletter list
- Inquiry management on portal
- User management — vendors and buyers

### Blog System
- Admin creates and publishes blog posts
- Built with Laravel Livewire for real-time editing
- Category-based blog organization
- Public blog listing and detail pages

### Inquiry System
- Buyer submits inquiry on product page
- Inquiry logged in admin portal
- Instant WhatsApp notification to vendor
- Dual tracking — portal + WhatsApp

## 🛠 Tech Stack

| Layer | Technology |
|---|---|
| Backend | Laravel, PHP |
| Frontend | Laravel Livewire, Blade, Bootstrap, JavaScript |
